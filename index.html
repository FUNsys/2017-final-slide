FUNsysチーム

FUNsysチームについて

  • 発足：2017年4月
  • 目的：未来大学内のデータを有効活用してアプリ開発をしたいと思っている未来大生を支援する
    •

    背景

  • 未来大学は情報系の大学 → アプリ開発をしている学生が一定数存在
  • 大学のデータを扱ってアプリ開発をしたいというニーズが存在する
    •

    現状

  • 大学の時間割やシラバスの情報を入手するには、PDFやHTMLを解析する等の方法しかない
    •

    コンセプト

  • 未来大にあるデータの
    アクセサビリティを向上させる
  • 自発的にアプリ開発を行う、
    未来大生開発者向けのサポートを行う
    •

    FUN-API

  • 未来大のデータを提供するWeb API
  • 現在は 時間割・教員情報・教室情報を提供
  • アクセスには学内のメールアドレスで
    認証する必要がある
    •

    FUNカレンダー

  • APIを使った、未来大生の為のカレンダーアプリ
  • 時間や教室、教員等の情報から必要な情報を
    入れ替える事が出来るのが特徴
